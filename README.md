## 💻 Getting Started <a name="getting-started"></a>


To get a local copy up and running, follow these steps.

### Prerequisites

In order to run this project you need:


```sh
 gem install rails
 git
 yarn
 Nodejs
```


### Setup

Clone this repository to your desired folder:


```sh
  cd my-folder
  git clone git@github.com:d4nQw3rty/hello-react-front-end.git
```


### Install

Install this project with:


```sh
  cd my-project
  npm install
```


### Usage

To run the project, execute the following command:


```sh
  npm start
  if you want to run the project in a different port you can run
  PORT=3001 npm start, recommended.
```


### Run tests

To run tests, run the following command:


```sh
    npm test
    if you want to run the tests in a different port you can run
    PORT=3001 npm test, recommended
```


### Deployment

You can deploy this project using:



```sh
    npm run build
```
